avoid ordering or usage errors.

## 4. Standards Notes

- **ISO laydown threading inserts (IR/ER)** are for UN, UNF, and metric threads
- **SER holders** accept **ER** external threading inserts
- **API chasers** are for **American Petroleum Institute oilfield thread forms**
- API chasers are **not interchangeable** with ISO laydown systems
- Items marked **UNKNOWN** should be verified before re-ordering
