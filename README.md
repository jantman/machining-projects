## Drill / Tap / Screw Charts

Comprehensive tap drill, clearance drill, and counterbore/countersink charts for inch screws (#0-#10, #12, 1/4"-1"), including socket head cap screws (SHCS) and flat head cap screws (FHCS). Available in multiple formats:

* [inch_taps_drills-letter.pdf](inch_taps_drills-letter.pdf) - Letter size (8.5" × 11"), landscape orientation, single page
* [inch_taps_drills-tabloid.pdf](inch_taps_drills-tabloid.pdf) - Tabloid size (11" × 17"), landscape orientation, single page
* [inch_taps_drills-tabloid2x2.pdf](inch_taps_drills-tabloid2x2.pdf) - Tabloid size, 2×2 grid (34" × 22" when taped together)
* [inch_taps_drills-tabloid3x3.pdf](inch_taps_drills-tabloid3x3.pdf) - Tabloid size, 3×3 grid (51" × 33" when taped together)
* [inch_taps_drills-24x36.pdf](inch_taps_drills-24x36.pdf) - Plotter size (24" × 36"), landscape orientation, single page
* [inch_taps_drills-36x48.pdf](inch_taps_drills-36x48.pdf) - Plotter size (36" × 48"), landscape orientation, single page
